# FactBank — Papers

Five write-ups of the FactBank project, by Mohannad (drafted with an AI assistant). Start with the
**Research Report** for the full story, or the **Idea Paper** for the short version. Each PDF has an
editable `.docx` source alongside it.

| paper | what it covers |
|---|---|
| [1. Idea Paper](FactBank-1-Idea-Paper.pdf) | the concept, motivation, and real-world use, in plain terms |
| [2. Technical Paper](FactBank-2-Technical-Paper.pdf) | architecture, retrieval, data flow, and where the gains come from |
| [3. Verdict Paper](FactBank-3-Verdict-Paper.pdf) | a critical assessment: strengths, weaknesses, novelty, maturity, risks |
| [Research Report](FactBank-Research-Report.pdf) | the full technical report and foundation for a future paper |
| [Evidence Ledger](FactBank-Evidence-Ledger.pdf) | every claim mapped to a real file / number in the repo (F-001…F-065) |

Also in this folder: `FACTBANK-DRAFT.md` (an earlier working draft) and `template-brain-report.md` (an
earlier report on the in-GGUF bank). GitHub renders the PDFs in-page when you click them.
